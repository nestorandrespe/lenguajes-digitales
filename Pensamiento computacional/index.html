Repeticiones

  • Una máquina no se cansa de repetir acciones
  • Varios sistemas consisten en repetir una y otra vez una acción, por ejemplo la animación

Multi dimensional

  • El ser humano le cuesta pensar en varias dimensiones. Nuestra solución: vectores.
  • Una máquina puede ejecutar cálculos o vectorizaciones de manera super eficiente.
  • Los vectores pueden tener n dimensiones y se usan para procesamiento de imágenes, simulaciones y machine learning.
